## 🎯 What is Harmonix?

Harmonix is a simple application designed to help you organize and enjoy your music collection. It works smoothly on Windows and aims to make your music listening easier. You don’t need any technical skills to use this software.

With Harmonix, you can:

- Manage your music files quickly.
- Create and save playlists.
- Play music with easy controls.
- Support common audio formats like MP3, WAV, and FLAC.

To run Harmonix on your computer, make sure you have:

## ⚙️ Basic Features Explained

Here is how to use some key features in Harmonix:

### Adding Music to the Library

1. Click the “Add Music” button.
2. Select one or more folders on your computer.
3. Harmonix will scan for music files and add them to your library.

### Playing Songs

- Select a song from your library or playlist.
- Click the “Play” button.
- Use the pause and skip buttons to control playback.

### Creating Playlists

1. Click “New Playlist”.
2. Give the playlist a name.
3. Drag songs from your library into the playlist.
4. Save the playlist for later use.
